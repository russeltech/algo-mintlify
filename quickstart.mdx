AlgoSeek gives you access to 200+ institutional-grade datasets spanning equities, options, futures, and more. This guide walks you through logging into the Console, picking a dataset, and retrieving data — whether via ArdaDB SQL, file delivery, or API — so you can go from credentials to querying in minutes.

<Steps>

  <Step title="Log In to the Console">
    Navigate to [console.algoseek.com/auth/login](https://console.algoseek.com/auth/login) and enter the credentials provided when your subscription was activated. Once logged in, your dashboard displays your active subscriptions, delivery settings, and full data catalog access. If you haven't received credentials yet, contact [sales@algoseek.com](mailto:sales@algoseek.com).
  </Step>

  <Step title="Browse the Data Catalog">
    In the left navigation, go to **Data → Data Sets**. Use the filters to narrow by asset class — **Equity**, **Options**, or **Futures** — and by data type: **Tick**, **Intraday Bar**, **Daily**, or **Reference**. Click any dataset card to open its full field documentation, schema reference, and downloadable sample data.
  </Step>

  <Step title="Choose Your Access Method">
    AlgoSeek supports four delivery modes. Select the one that best fits your workflow:

    | Method | Best For | Latency | Formats |
    |---|---|---|---|
    | **ArdaDB SQL** | Backtesting, research, ad-hoc queries | Seconds | SQL |
    | **File Delivery** | Bulk data, pipeline ingest | Hours / daily | CSV, Parquet, HDF5, FIX, Feather |
    | **REST API** | App integration, programmatic access | Near real-time | JSON |
    | **Co-Location Feed** | HFT, market making, low-latency trading | \<15 µs | Binary, TCP/Multicast |
  </Step>

  <Step title="Run Your First Query">
    Connect to **ArdaDB** using the connection string shown in your Console under **Delivery Settings → ArdaDB**. Then run the query below to pull Apple's minute-bar trade data for January 2024:

    ```sql
    SELECT
        TradeDate,
        Ticker,
        OpenPrice,
        ClosePrice,
        sum(Volume) AS TotalVolume
    FROM USEquityMarketData.TradeAndQuoteMinuteBar
    WHERE Ticker = 'AAPL'
      AND TradeDate BETWEEN '2024-01-01' AND '2024-01-31'
    GROUP BY TradeDate, Ticker, OpenPrice, ClosePrice
    ORDER BY TradeDate
    ```

    <Note>
      Results stream back in seconds even when scanning billions of rows. ArdaDB's columnar engine is purpose-built for time-series financial data at institutional scale.
    </Note>
  </Step>

</Steps>
